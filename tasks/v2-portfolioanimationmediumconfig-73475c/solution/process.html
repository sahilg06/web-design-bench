How we work

Four movements —
one choreography.

Every engagement moves through the same four movements: Discovery, Strategy, Design, and Motion. Timelines shift with scope, but the sequence is always the same — because clarity in the beginning is what makes the end feel effortless.

01
Weeks 1—2

Discovery

We start by listening. Workshops with founders, stakeholder interviews, audits of everything already in market, and a landscape map of adjacent brands. The output is a written brief with sharp constraints — the invisible scaffolding every downstream decision hangs on.

Kickoff workshop Stakeholder interviews Competitive audit Written brief
02
Weeks 3—5

Strategy

Positioning, narrative, and the tone-of-voice document. We define the emotional temperature of the brand and — critically for us — the motion principles that will govern how it behaves. Is it patient? Restless? Assertive? Meditative? Motion has grammar, and this is where we write it.

Positioning Narrative arc Tone of voice Motion principles
03
Weeks 6—10

Design

Type, color, logo, layout systems, and the first flat visual language. We work in tight two-week loops with the client, showing rough directions early rather than one polished reveal. Every visual element is designed to move — we never draw a mark we can't animate.

Visual identity Type & color Layout system Component library
04
Weeks 11—14

Motion

The final movement: choreographing everything. Logo behaviors, UI transitions, launch films, campaign loops, and — always — a motion guide the internal team can extend without us. We handoff production-ready assets, animation source files, and code where relevant.

Motion guide Logo behaviors Launch film Production handoff
