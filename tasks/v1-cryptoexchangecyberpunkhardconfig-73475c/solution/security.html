Architecture

Four-layer defense stack

Every request crosses four independent security domains before touching customer assets.

LAYER 01
◉

Network Perimeter

Cloudflare Enterprise WAF with DDoS mitigation up to 384 Tbps. Anycast BGP routing across 275 PoPs. Rate-limited at the IP and API-key layer with adaptive throttling. All traffic terminated at TLS 1.3 with certificate pinning.

LAYER 02
⌘

Application Isolation

Zero-trust service mesh (Istio + SPIFFE). Every microservice authenticates with rotating mTLS certificates. Order routing runs in a hermetic enclave with attested boot. No production credentials on-disk anywhere.

LAYER 03
❄

Wallet Segmentation

98.4% of assets in geographically distributed cold vaults with 4-of-7 multisig quorum. Hot wallets rebalanced every 15 minutes. Hardware security modules (FIPS 140-3 Level 4) for all signing keys.

LAYER 04
◆

Continuous Audit

Merkle-tree proof of reserves published every 6 hours to Ethereum L1. Independent audits by Trail of Bits, OpenZeppelin, and Halborn. Real-time on-chain reconciliation with public dashboards.
